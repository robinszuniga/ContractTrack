🔗 URL del servidor Apps Script
Para importar datos automáticamente del SECOP II necesitas desplegar el Apps Script una sola vez.
Descarga el archivo ContractTrack_AppsScript.gs, síguelas instrucciones incluidas y pega la URL aquí.
Pasos para desplegar el Apps Script (5 minutos):
1. Abre script.google.com con tu cuenta Google
2. Clic en "Nuevo proyecto" → pega el contenido del archivo .gs
3. Guarda (Ctrl+S) → nombre: ContractTrack Backend
4. Clic "Implementar" → "Nueva implementación" → Tipo: Aplicación web
5. Ejecutar como: Yo · Acceso: Cualquier persona
6. Clic "Implementar" → copia la URL → pégala arriba
